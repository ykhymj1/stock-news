## 🎯 핵심 기능

✅ **시장별 분리** - 🇰🇷 한국 / 🇺🇸 미국 탭으로 구분

✅ **추천 히스토리** - 날짜별로 추천했던 종목 모두 기록

## 🚀 빠른 시작 (3분)

### 1. 파일 배치
4개 파일을 한 폴더에 둡니다:
- `index.html`
- `app.js`
- `manifest.json`
- `sw.js`

### 2. 실행 방법 (3가지 중 선택)

#### 방법 A: 사무실 PC에 그냥 열기 (가장 쉬움)
- `index.html` 더블클릭 → 브라우저에서 열림
- ⚠️ 단점: PWA 기능(서비스워커, 홈화면 설치) 작동 안 할 수 있음

#### 방법 B: 로컬 서버로 실행 (권장)
사무실 PC 명령 프롬프트에서:
```cmd
cd C:\stockradar  (파일 둔 폴더)
python -m http.server 8080
```
브라우저에서 `http://localhost:8080` 접속

#### 방법 C: GitHub Pages로 배포 (이게 최고) ⭐
이미 AMT/KY 시스템 GitHub Pages 쓰고 계시니 동일한 방식:

1. GitHub에서 새 저장소 만들기 (예: `stockradar`)
2. 4개 파일 업로드
3. Settings → Pages → Source: `main` 브랜치 선택
4. 핸드폰/PC에서 `https://[username].github.io/stockradar` 접속

이렇게 하면 **Pixel 9에서 어디서나 접속 가능** + PWA로 홈화면 설치도 됩니다.

### 3. DART API 키 발급 (선택, 한국 공시 보려면)
- https://opendart.fss.or.kr 접속
- 우상단 **인증키 신청/관리** → **인증키 신청**
- 이메일 인증 → **카드 없이 즉시 발급** ✅
- 받은 키를 앱 → 설정 → DART API 키에 입력

> **DART 키 없어도 작동함** (한국 뉴스 RSS는 그대로 작동, 단 공시는 못 봄)

### 4. Cloudflare Worker URL 입력 (선택)
이미 `https://ykh-stock-proxy.kyunghoyou.workers.dev` 사용 중이시면 설정에 그대로 입력하시면 됩니다. 비워두면 AllOrigins 프록시로 폴백.

## 🎮 사용법

### 첫 사용
1. 앱 열면 **🇰🇷 한국 시장** 탭이 기본
2. 상단의 **"🔍 지금 분석"** 버튼 클릭
3. 30초~1분 후 TOP 5 + 추가 후보 5개 표시됨
4. 종목 카드 누르면 상세 분석 모달 열림

### 종목 추적 시작
1. 관심 종목 카드 클릭 → 상세 모달
2. 하단 **"📌 추적 시작"** 버튼 클릭
3. 추천 시점 가격 자동 저장됨
4. **📈 추적** 탭에서 모든 추적 종목 + 그래프 확인

### 추적 갱신
- **수동**: 추적 탭 → "↻ 전체 시세 갱신"
- **자동**: 설정 → 자동 갱신 주기 (예: 10분)

### 히스토리
- **📅 히스토리** 탭에서 날짜별 추천 기록
- 추적 중인 종목은 자동으로 수익률 표시됨

### 삭제
- **개별**: 추적 탭 → 종목 카드 → "삭제"
- **전체**: 추적 탭 → "🗑 전체 삭제"
- **모든 데이터 초기화**: 설정 → 전체 초기화

## 🔍 추천 알고리즘 (어떻게 선정되는지)

### 1단계: 데이터 수집
- **🇰🇷**: DART 공시 (있으면) + 한경/매경 RSS
- **🇺🇸**: SEC EDGAR 8-K + Yahoo Finance 뉴스 (10개 주요 종목)

### 2단계: 종목 추출
- 한국: 회사명 → 종목코드 매핑 (30개 주요 종목)
- 미국: 회사명/티커 매핑 (15개 주요 종목)

### 3단계: 임팩트 점수 (1~10)
키워드 기반 분석:
- **+3점 키워드**: "단일판매ㆍ공급계약체결", "어닝 서프라이즈", "FDA 승인", "사상 최대" 등
- **+1점 키워드**: "상향", "확대", "신제품" 등
- **-3점 키워드**: "거래정지", "임상 실패", "어닝 쇼크" 등
- **-1점 키워드**: "하향", "축소" 등

기본 5점에서 키워드별로 가감산 → 최종 1~10점

### 4단계: 종목별 점수 합산
- 한 종목에 여러 뉴스 있으면 점수 누적
- 최고 임팩트 점수 + 뉴스 개수 고려

### 5단계: TOP 10 정렬
- 누적 점수 기준 내림차순
- 상위 5개 = TOP 5 (강조 표시)
- 6~10위 = 추가 후보

## 🆘 문제 해결

### "RSS/API 응답이 없습니다" 에러
- AllOrigins 프록시가 일시적으로 응답 지연될 수 있음
- 잠시 후 다시 시도
- 또는 DART API 키 입력 (한국 공시 우선 사용)

### 종목 카드가 너무 적게 나옴
- 설정 → 최소 임팩트 점수를 5로 낮추기
- 또는 한국 시장은 장중(9~16시)에만 활발함

### "시세 가져오기 실패"
- Worker URL이 정확한지 확인
- 또는 비워두고 AllOrigins 폴백으로 시도
- 미국 종목은 한국 시간 23:30 이후에 가장 정확

### 사파리에서 작동 안 함
- iOS Safari는 일부 기능 제한 있음 (특히 LocalStorage)
- Chrome/Edge 권장

### 데이터가 사라짐
- LocalStorage 사용 → 브라우저 캐시 삭제 시 함께 삭제됨
- **주기적으로 설정 → 데이터 내보내기로 백업** 권장

## ⚠️ 주의 사항 (반드시 읽으세요)

이 앱은 **정보 제공 도구**일 뿐, 투자 자문이 아닙니다:

1. **추천은 100% 키워드 기반 자동 분석** - AI가 시장을 예측한 게 아님
2. **"추천 = 매수 신호" 아님** - 참고용 정보
3. **시장은 뉴스대로 움직이지 않을 때가 많음** - 좋은 뉴스에 갭다운 빈번
4. **Pump & Dump 위험** - 특히 소형주는 가짜 뉴스로 펌핑 가능성
5. **무료 데이터의 한계** - 30~60초 지연, 일부 누락 가능
6. **본인 판단으로 투자** - 손실에 대한 책임은 본인

**투자는 본인 판단이며, 이 앱은 정보를 빠르게 보여주는 도구일 뿐입니다.**
